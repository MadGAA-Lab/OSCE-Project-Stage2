```
- [...]

- [...]

- [...]

- [...]

```

---

### Example

```
Feature: Automated Report Generation

Trigger Conditions:
- User uploads a supported dataset
- User selects at least one analysis module

Behavior:
- Validate dataset format and structure
- Process using selected analysis modules
- Generate structured report assets

Input Requirements:
- Supported formats: CSV, JSON
- Maximum file size: 50MB

Output Requirements:
- PDF report
- Summary JSON file
- Visual charts and metrics

Error Handling:
- Unsupported format → Display validation error
- Processing timeout → Retry and notify user

User Interaction:
- Progress indicator during analysis
- Ability to cancel processing

Dependencies:
- Cloud storage service
- Background processing service
```
